# USA Rent Analysis

Proyecto para limpieza, segmentación y análisis de datos de la renta 
en estados unidos con datos de portales de alquiler entre los años 2000-2018.

## Estructura

- `src/`: lógica de procesamiento reutilizable
- `analysis/`: notebooks de análisis exploratorio
- `data/`: datos crudos y procesados
- `main.py`: script ejecutable
